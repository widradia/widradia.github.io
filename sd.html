Kegiatan Pembelajaran

SD Santo Andreas mengadopsi kurikulum berbasis Fun Leraning. dengan metode Fun Learning siswa menjadi gembira dalam memahami yang disampaikan oleh guru mereka dan semakin memahaminya. metode FUN Learning ini didukung beberapa kegiatan seperti:

  • Finding
  • Utering
  • Nailing
  • Mnemonik
  • Parent Go To Class
  • Project Based Learning
